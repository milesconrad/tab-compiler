# tab-compiler
This is a python script that takes notes as input, and converts them to guitar tablature.

# expected input and output
notes: `A G F# D D C# A`

fret: `7`

output: <br>
`---------------`<br>
`-----7---------`<br>
`---0-------6---`<br>
`-------0-0-----`<br>
`-0-----------0-`<br>
`---------------`<br>
